# Farm to Retail Price Analysis
This project explore how meat values change across the supply chain from farm to wholesale to retail for pork, beef, and chicken between 1970 and 2025. All data were sourced from the U.S. Department of Agriculture (USDA).

## Features
- Histogram visualizations
- Scatter plots
- Interactive Streamlit dashboard

## Technologies
- Python
- Pandas
- Plotly
- Streamlit
